## 🔧 技术细节

### .gitignore 规则详解

```gitignore
# ✅ 会上传
src/          # 所有源代码和示例
*.py          # Python 源文件

# ❌ 不会上传
tests/        # 测试代码
__pycache__/  # Python 缓存
.venv/        # 虚拟环境
.idea/        # IDE 配置
```

### 为什么示例移到 src/？

```python
# 示例文件需要这样导入：
import dsvis

# 如果在 examples/ 目录，需要 sys.path 操作：
import sys
sys.path.insert(0, '..')
import dsvis  # 不优雅！

# 如果在 src/ 目录，直接导入即可 ✓
```
